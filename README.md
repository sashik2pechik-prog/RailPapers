**RailPaper** is the premier digital archive and leaderboard hub for **Dead Rails** (Roblox) speedrunners. We document the fastest times, verify records, and build a community for those who master the iron tracks.

## 🛠️ Key Features
* **Global Leaderboards:** Real-time tracking of the world's fastest conductors.
* **The Railway Code:** A comprehensive set of rules to ensure fair play.
* **Archive System:** A "newspaper-style" database of historical runs and community milestones.
* **24/7 Hosting:** Powered by Vercel for instant access worldwide.

## 📜 The Code (Short Version)
1. **Video Proof:** Every submission must include a link to a full, unedited run.
2. **No Cheats:** Use of scripts, exploits, or external software results in a permanent ban.
3. **Accuracy:** You must state the game version and train type used.
4. **Conduct:** Respect all passengers in the forum. No toxicity allowed.

## 🚀 How to Submit a Run
Currently, submissions are handled via GitHub. To submit your time:
1. Open an **Issue** with the title "New Record Submission".
2. Provide your **Roblox Username**, **Time**, and **YouTube Link**.
3. A moderator will review the footage and update the `index.html`.

## 📬 Contact & Support
Maintained by the **RailPaper Administration**. If you find a bug on the site, please open an issue in this repository.
